# SQL Window Functions Practice Project

## 📌 Project Overview

This project is a collection of **real-world SQL Window Function exercises** built around a **Hospital Management Dataset**. The goal is to strengthen advanced SQL skills by solving analytical business problems using ranking, running totals, moving averages, cumulative calculations, comparative analysis, and advanced window functions.


## 🎯 Objectives

This project demonstrates how SQL Window Functions can be used to:

- Rank business entities
- Calculate running totals
- Build cumulative metrics
- Analyze month-over-month growth
- Compute moving averages
- Compare current vs previous values
- Segment data into groups
- Calculate contribution percentages
- Perform advanced healthcare analytics

# 🎯 Learning Outcomes

After completing this project, you will be able to:

- Understand how SQL Window Functions work
- Solve real business reporting problems
- Create analytical SQL reports
- Build running totals and moving averages
- Perform month-over-month growth analysis
- Rank business entities efficiently
- Compare historical values using LAG()
- Use FIRST_VALUE() and LAST_VALUE() effectively
- Segment datasets using NTILE()
- Write advanced SQL suitable for interviews and production environments
